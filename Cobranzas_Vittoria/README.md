# Cobranzas_Vittoria API Final

API .NET 8 con arquitectura por carpetas (n-capas dentro de un proyecto):
- Controllers
- Services
- Repositories
- Interfaces
- DTOs
- Entities
- Data
- Middleware

## Requisitos
1. Ejecutar primero `Vittoria_DB_Completo.sql`
2. Ajustar `appsettings.json` si tu SQL Server usa otra conexión
3. Ejecutar:
   - dotnet restore
   - dotnet run

Swagger:
- http://localhost:5000/swagger
